# PayOne PCP JavaScript Client SDK

This repository contains the PayOne PCP Client SDK for JavaScript.  
A demo App is also provided in the `demo` directory.

### Build the library

```sh
npm run build
```

### Build the demo app

```sh
npm run build:demo
```

### Run the demo app

```sh
npm run start:demo
```
